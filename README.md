# Agentic Workspace: Adswerve Asana Integration

Welcome! This repository is configured to be developed by AI. You don't need to know how to write code to build features here, you just need to act as the orchestrator. 

## First-Time Setup
1. Open this folder in **Google Antigravity**.
2. Open the Chat / Mission Control and type: `Run @project-onboarder`.
3. The AI will ask you for some Asana App credentials. Provide them.
4. Restart Antigravity.
5. Type `Run @asana-mcp-tester` to verify your connection. A browser window will pop up asking you to authorize Asana. Click allow.

## Daily Workflow
Building software here happens in two steps:

**1. Start Work**
Type `/asana-start` in the chat. The AI will read your Asana backlog, ask you what you want to work on, create a safe workspace (a git branch), and plan the code. Tell it to execute!

**2. Finish Work**
When you are happy with the feature, type `/asana-sync` in the chat. The AI will automatically save your code, push it to the cloud, create a PR link, write a status update for your Project Manager, and close the Asana ticket.
